Коротко о проекте

Данный проект является концепцией сервиса, который предоставляет информацию о ресторанах города,

так же предоставляет возможность бронирования столика.

Для владельцев ресторана реализована возможность регистрации собственного заведения,c возможностью размещения в сервисе.

Реализовано получение заявок на бронирование от пользователей. Техническая поддержка и уведомления от самого сервиса присутствуют

Имеется возможность работы с api.

Работа с api запросами имеет разделение доступа для POST PATCH PUT И DELETE запросов

Для владельца сервиса добавлен Телеграм Бот через который можно получать информацию из БД

Цели

Была единственная цель, продемонстрировать навыки по работе с Django framework, ORM, DRF, HTML и СSS.
