Ключ доступа-Токен

Токен выдается только после регистрации пользователя.

В первую очередь вам необходимо пройти регистрацию,

это можно осуществить 2 способами:

1. Через ui интерфейс

2. Через POST запрос

POST запрос можно реализовать 2 способами

1. Через браузер-после основного url http://127.0.0.1:8000/ Используйте метод "users/"

В интерфейсе введите данные, на этом все

________________________________________

2. POST запрос через REST CLIENT

Создайте POST запрос:

POST http://127.0.0.1:8000/users/

Передайте тип данных:

Content-Type: application/json

Отправьте данные:

{

"username": "Логин",

"email": "Почта",

"password": "Пароль",

"first_name": "Имя",

"last_name": "Фамилия"

}

Запрос токена возможен только через личный кабинет.

Выдача токена производится исключительно Администратором.

Где получить Токен:

Личный кабинет-профиль и почта-запросить токен

Токен будет направлен в раздел "уведомления"

GET запросы

GET запросы на получение информации о ресторанах,

не требуют токена.

GET запрос можно реализовать 2 способами

1. Через браузер

2. Через REST CLIENT

_____________________________________________

1. Через браузер-после основного url http://127.0.0.1:8000/

Используйте метод "restaurant_api/"

____________________________________

2. GET запрос через REST CLIENT

Создайте GET запрос:

GET http://127.0.0.1:8000/restaurant_api/

Передайте тип данных:

Content-Type: application/json

Для поиска по тексту используйте любой из методов:

"Name_restaurant": "Название,

"Address": "Адрес",

"rating": Рейтинг,

"about":"Описание"

"terrace_restaurant": "Есть,Нет",

"parking_restaurant": "Есть,Нет",

"kitchen": "Кухня"

"average_check": Средний чек,

"email": "почта@gmail.com",

"phone": "номер",

"to_publish": true,false

______________________________________________

Пример: http://127.0.0.1:8000/restaurant_api/?rating=5

Поиск ограниченного кол-ва записей LIMIT&OFFSET

Используйте метод:/?limit=кол-во&offset= c какой записи начать поиск

Пример: GET http://127.0.0.1:8000/restaurant_api/?limit=2&offset=4
